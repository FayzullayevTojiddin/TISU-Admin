Privacy Policy

Last updated: January 10, 2026

This Privacy Policy explains how information is handled when using the TISU Teacher Attendance App, developed for teachers of TISU University.

1. Information We Collect

The application collects only the minimum data required for authentication and attendance functionality.

Personal Information:

  • First name
  • Last name
  • Password (used only for login authentication)

Attendance Data:

  • Attendance records of students (present / absent)
  • Selected faculty, group, and classroom (retrieved from the university database)
  • Optional photo upload at the end of attendance

2. Information We Do Not Collect

  • Location data (GPS)
  • Phone number or email address
  • Contacts
  • Device identifiers
  • Background activity data
  • Audio recordings

3. How We Use Information

  • User authentication (teacher login)
  • Displaying relevant faculties, groups, and classrooms
  • Managing and recording student attendance
  • Attaching an optional image as attendance confirmation

4. Photo Uploads

Photo uploads are optional and are performed only when the user chooses to upload a photo. Photos are used solely for attendance confirmation. No biometric or facial recognition processing is performed.

5. Data Storage and Security

All data is stored securely on the university’s backend servers. Passwords are stored in encrypted form. No data is shared with third parties.

6. Third-Party Services

This application does not use third-party analytics, advertising, or tracking services.

7. User Rights

  • Request access to personal data
  • Request correction or deletion of account data

8. Children’s Privacy

This application is intended only for authorized university staff and is not designed for use by children.

9. Changes to This Privacy Policy

This Privacy Policy may be updated if application functionality changes. Any updates will be published on this page.
